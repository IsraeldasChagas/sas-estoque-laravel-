✅ IMPLEMENTADO:

3 FILTROS LADO A LADO:
  📅 Mês/Ano        🏢 Unidade        📊 Status        🔄 Limpar


🎯 FILTROS DISPONÍVEIS:

1️⃣ FILTRO POR MÊS/ANO 📅
   → Últimos 12 meses + "Todos os boletos"
   → Filtra pela data de vencimento

2️⃣ FILTRO POR UNIDADE 🏢
   → Carregado automaticamente da API
   → "Todas as unidades" + suas unidades

3️⃣ FILTRO POR STATUS 📊
   → Todos os status (padrão)
   → A vencer
   → Vencido
   → Pago
   → Cancelado

4️⃣ BOTÃO LIMPAR 🔄
   → Remove todos os filtros
   → Volta para "Todos"


🚀 COMO USAR:

1️⃣ FILTRO SIMPLES:
   a) Selecione apenas um filtro
   b) Exemplo: Status = "Vencido"
   c) ✅ Mostra apenas boletos vencidos

2️⃣ FILTROS COMBINADOS:
   a) Selecione múltiplos filtros
   b) Exemplo:
      • Mês: Janeiro 2026
      • Unidade: Matriz
      • Status: Pago
   c) ✅ Mostra apenas:
      - Boletos de Janeiro 2026
      - Da unidade Matriz
      - Que estão pagos

3️⃣ LIMPAR FILTROS:
   a) Clique no botão "🔄 Limpar Filtros"
   b) ✅ Todos os filtros voltam para "Todos"
   c) ✅ Mostra todos os boletos
