📋 COMO TESTAR AGORA:

1️⃣ REINICIE O SERVIDOR LARAVEL:
   - Pare o servidor (Ctrl+C)
   - Execute: cd backend
   - Execute: php artisan serve --host=0.0.0.0 --port=5000

2️⃣ ABRA O SISTEMA:
   - http://localhost:5000
   - Faça login com usuário ADMIN

3️⃣ TESTE CRIAR USUÁRIO:
   - Clique em "Usuários" no menu
   - Clique em "+ Novo Usuário"
   - Preencha:
     ✓ Nome: João Silva
     ✓ Email: joao@exemplo.com
     ✓ Senha: 123456
     ✓ Confirmar Senha: 123456
     ✓ Perfil: ASSISTENTE_ADMINISTRATIVO
     ✓ Status: Ativo
   - Clique em "Salvar"
   - ✅ DEVE CRIAR O USUÁRIO!

4️⃣ TESTE UPLOAD DE FOTO:
   - Clique em "+ Novo Usuário"
   - Preencha os dados
   - Clique em "Escolher foto"
   - Selecione uma imagem JPG ou PNG
   - Pré-visualização aparece
   - Salve
   - ✅ DEVE SALVAR COM FOTO!

5️⃣ TESTE EDITAR USUÁRIO:
   - Na lista, clique em "✏️ Editar"
   - Mude o nome
   - Salve
   - ✅ DEVE ATUALIZAR!

6️⃣ TESTE TROCAR FOTO:
   - Edite um usuário
   - Clique em "Trocar"
   - Escolha nova foto
   - Salve
   - ✅ DEVE TROCAR A FOTO!
